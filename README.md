# Nova Starter Website

This folder is a clean, modern, responsive starter website (no framework).

## Run locally

Option A (works everywhere): open the file directly:

```bash
open website/index.html
```

Option B (recommended if you want a local server):

```bash
cd website
ruby -run -e httpd . -p 5173
```

Then open `http://localhost:5173`.

## Customize

- Edit content in `index.html`
- Change styles in `styles.css`
- Update behavior in `script.js` (especially `EMAIL_TO`)
